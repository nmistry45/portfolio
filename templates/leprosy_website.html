003 : Leprosy Patient's Digital Database Website

Simple & Easy website created for digitizing Leprosy patient's file records.

"The Biggest Disease Today Is Not Leprosy or Tuberculosis, But Rather The Feeling of Being Unwanted"

Mother Teresa

The digitizing of manual data is always a tough task as the system should be really simple to understand but equally capable of handling any kind of input and must not face any error. Add an additional level of difficulty of Medical Data, & this makes it a really tough as the words, the kind of medical data that gets stored has to be accurate & error-less. This website was created with the users in mind, & is a collaboration between my college - Shah & Anchor Kutchhi Engineering College with renowned NGO Bombay Leprosy Project and The International Leprosy Mission: Bangladesh. More details about this project can be found at my GitHub.
