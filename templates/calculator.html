008 : Scientific Calculator using Arduino

Using Keypad as input & performing simple as well as complex calculations that ultimately get displayed on an LCD Screen.

"A Lot Of People... Use A Calculator!"

Joe Strummer

Everybody needs a calculator at some point in their life. It is an integral part of everyone's life. In this project, I explored the "Behind-The-Scenes" functioning of the calculator. Using a Keypad & generating the logic was immensely fun & this project was also a part of my Internship at a Robotics Company. More details about this project can be found on my GitHub.


GitHub Link:- https://github.com/nmistry45/Calculator-Scientific-Calculator-Using-Arduino
