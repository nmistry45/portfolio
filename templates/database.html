002 : Distributed Database Management System

Made a Distributed Database Management System using Core Java and GCP.

“Database: The Information You Lose When Your Memory Crashes”

Dave Barry

A database management system (DBMS) is very essential to any tech company. Whether it be SQL or NoSQL, depending on the use case, it is really important to have a comprehensive DBMS. And with the growth of Cloud Computing, a Distributed DBMS that is spread across the cloud and local machines is a huge plus point. In this project, we have implemented a Distributed DBMS using Core Java and Google Cloud Platform (GCP) while also using all the essential good practices of Java programming like using enums and correct naming conventions.
