001 : Invoicify

Helping businesses handle their invoices more efficiently and send them to any customer as a PDF through email.

“If you want to be successful, you must pay the price in advance.”

Godwin Elendu
Many business owners are always facing the issue of handling their invoices and making sure to remind their customers of an upcoming payment deadline. Invoicify is a MERN stack-based Cloud Web application that helps businesses and their owners to create invoices easily and keep a track of them through the intuitive dashboard with charts and easy navigation to the customers. The backend or server-side code was created entirely using Amazon Web Services (AWS) services such as CloudFormation, Elastic Beanstalk, Lambda, SQS, S3, Cognito, DynamoDB, API Gateway, CloudWatch, and EC2. This helped keep the application scalable even in the future.


GitHub Link:- https://github.com/nmistry45/invoicify
