002 : Stock Price Prediction using LSTM

Predicting stock prices of Tata Global Beverages, Apple, Microsoft, Tesla, and Facebook.

“We don't have to be smarter than the rest, we have to be more disciplined than the rest ”

Warren Buffett
The stock market is a very demanding and ever-changing field for predictions. The prices are constantly shuffling around and no one can make a perfect judgment for a particular stock of a company. But, using machine learning algorithms, we can almost predict the correct stock prices or closing stock prices for the shares. Also, as the stock market is changing every day, it is a time-series dataset and requires special algorithms like Recurrent Neural Networks (RNNs) to work properly. Hence, we implemented a Long Short-Term Memory (LSTM) based neural network to solve this prediction problem.


GitHub Link:- https://github.com/nmistry45/stock-price-prediction-using-lstm
