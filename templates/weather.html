006 : Weather Monitoring System using Arduino

Monitoring Weather using multiple sensors & displaying the output on LCD Screen or smartphone using Bluetooth.

"It Is Best To Read The Weather Forecast Before Praying For Rain"

Mark Twain

This system consists of multiple sensors interfacing with a micro-controller board & data getting shared. It is an amazing project that I loved exploring & creating. The hardware side of engineering fascinated me and this project was a part of my Internship at a Robotics Company. More details about this project can be found at my GitHub.
